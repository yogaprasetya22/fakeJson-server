## Getting started

Install JSON Server dan Faker
```sh
npm install -g json-server
npm i faker
```

ubah untuk menjanlankan json-server di package.json
```json
"scripts": {
    "start": "json-server --watch <nama_file>.json"
  }
```
  
  run
  ```sh
  yarn start
  ```
